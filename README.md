## Project Overview
This project analyzes retail customer purchase data to understand buying behavior, spending patterns, and product preferences. The aim is to convert raw transactional data into meaningful insights that support data-driven business decisions.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (PostgreSQL/MySQL)
- Power BI
  

## Project Workflow
1. Data Cleaning & Preparation (Python)
   - Handling missing values
   - Column formatting
   - Feature engineering

2. Data Analysis (SQL)
   - Revenue and spending trends
   - Customer segmentation
   - Discount and subscription analysis
   - Product performance queries

3. Visualization (Power BI)
   - Interactive dashboards
   - Sales and revenue charts
   - Customer and product insights

## Key Insights
- Loyal customers contribute higher revenue
- Discounts influence purchase frequency
- Subscribers show better long-term value
- Certain products consistently perform well
